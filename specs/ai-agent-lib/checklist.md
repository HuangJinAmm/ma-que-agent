# AI Agent Rust Lib 验收检查清单

## 1. 文档与工程基础

- [ ] 项目以 Rust `lib` 形式交付，而非仅二进制程序
- [ ] crate 根模块与公共导出清晰
- [ ] README 包含最小接入示例
- [ ] 对外 API 有基本文档说明
- [ ] 关键模块具备错误类型与返回语义说明

## 2. 配置系统

- [ ] 支持通过 TOML 定义主 agent
- [ ] 支持通过 TOML 定义 sub agent
- [ ] 支持通过 TOML 定义 tools
- [ ] 支持通过 TOML 定义 MCP server
- [ ] 支持通过 TOML 定义 workflow
- [ ] 支持通过 TOML 定义记忆 store 与记忆策略
- [ ] 支持默认值继承和局部覆盖
- [ ] 支持环境变量插值
- [ ] 非法配置可返回可定位的错误信息

## 3. `genai` 集成

- [ ] 模型调用统一基于 `genai` 实现
- [ ] 支持普通 chat 调用
- [ ] 支持流式响应
- [ ] 支持 tool calling
- [ ] 支持基础模型参数配置
- [ ] 支持输出 token 使用量或相关调用元数据

## 4. Agent 与 Sub Agent

- [ ] 可按名称获取并执行主 agent
- [ ] 主 agent 可路由到 sub agent
- [ ] sub agent 可继承并覆盖配置
- [ ] agent 支持私有记忆与共享记忆作用域
- [ ] 可限制最大递归深度
- [ ] 可限制最大委派次数
- [ ] 路由失败和执行失败有明确错误分类

## 5. Tools 注册与管理

- [ ] 存在统一 `Tool` 抽象
- [ ] 存在统一 `ToolRegistry`
- [ ] 支持静态注册 tools
- [ ] 支持运行时注册 tools
- [ ] 支持按 agent 范围筛选可见 tools
- [ ] 支持冲突检测或命名空间隔离
- [ ] 工具执行支持超时和并发限制
- [ ] 工具调用有日志或指标记录

## 6. MCP 能力

- [ ] 支持通过 TOML 配置 MCP server
- [ ] 至少支持 stdio 类型 MCP server
- [ ] 支持 MCP server 生命周期管理
- [ ] 支持 MCP tool 发现与缓存
- [ ] 支持将 MCP tool 映射为统一 tool 接口

## 7. Claude Skills 能力
